# Nucleo-F401RE GPIO

## Ψευδώνυμα ακροδεκτών (Pin Aliases)

| ψευδώνυμο   | ακροδέκτης |
| :---        | :---       |
| red_led     | 13         |
| user_led    | 13         |
| ld2         | 13         |
| builtin_led | 13         |

## GPIO (Είσοδος/Έξοδος γενικής χρήσης)

Ακροδέκτης 13: Λυχνία LED χρήστη (LD2)
- Έξοδος, ενεργή σε υψηλή κατάσταση (active high)
- PA5 στον μικροελεγκτή STM32F401
